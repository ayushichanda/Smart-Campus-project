# Smart Campus Energy Dashboard

## Overview

The **Smart Campus Energy Dashboard** is a monitoring system designed to track and analyze electricity consumption across different rooms and laboratories within a campus.
The system collects energy-related data such as **temperature and power usage** and visualizes it through an interactive dashboard.

This project helps identify **energy wastage**, detect abnormal power usage, and support better energy management in a smart campus environment.

---

## Features

* Real-time monitoring of **power consumption**
* **Room-wise energy tracking** for campus facilities
* **Temperature monitoring** of different rooms
* **Energy waste detection system**
* Interactive **data visualization using charts**
* Dashboard showing **alerts and system status**

---

## Dashboard Components

### 1. Energy Monitoring Table

Displays real-time data including:

* Room name
* Temperature (°C)
* Power consumption (W)
* Status (Normal, Alert, Energy Waste)

### 2. Temperature Trend

A line chart showing temperature variations across campus rooms.

### 3. Energy Waste Detection

A bar chart identifying rooms with high energy wastage events.

### 4. Power Consumption Analysis

A bar chart comparing electricity usage across different campus facilities.
